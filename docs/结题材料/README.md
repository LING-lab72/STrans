## 当前材料口径

- 系统形态：React 监控大屏 + FastAPI 服务 + 本地视觉模型 + SQLite 数据闭环。
- 视频来源：多路 RTSP/MJPEG、手机网络视频、本地图片和本地录像。
- 核心识别：车辆检测与跟踪、车牌识别、道路异常识别、道路区域分割。
- 派生分析：速度估计、白名单决策、道路坐标映射、禁停判断、拥堵热力图。
- 管理闭环：账号权限、摄像头管理、历史记录、告警证据与处置、模型参数、自适应调度、审计和智能报告。
- 配置工具：管理员可从功能中心打开项目内置道路建模页，生成 `road_logic_modeler.v1` 道路拓扑与摄像头标定 JSON。

## 使用顺序

最终交付先阅读 `19-结题交付总览.md`；正式打印、归档和提交优先使用 `output/pdf/latex-final/` 中的 PDF，正文维护以 Markdown 为事实源、以 `latex/` 为排版工程。再按角色选择 V2.0、软件说明书、测试报告、总结报告或 M01–M13 专项材料。答辩使用 `14-初始要求追踪与答辩问题链.md` 和 `ppt-html/`。每一项"已实现"能力都应能在代码、接口、页面、自动化测试或真实识别图中找到证据。
